Unreal Engine 4 Platformer


<     >

I believe strongly in learning-by-doing.

That is why, when I first heard of the performance, portability and graphical innovations that have been reached in Unreal 4, I wasted no time in getting right into learning it.

The screenshots you see are taken from a prototype cross-platform application I made in Unreal 4 through exploration, trial and error, and research. Java quickCheck Application


<     >

The best way to learn is by pushing your boundaries.

That is why, despite being less than comfortable with Java’s GUI functionalities, I dove right in to a very ambitious project that would bring together complex algorithms, massive data retrieval, and a custom-made user interface. Because I was so far out of my comfort zone, I found myself encountering problems I’d never even seen before. Yet through diligence, research, hands-on application and good old trial-and-error, I not only completed what I’d set out to achieve, but learned a number of invaluable skills along the way.

This app makes use of a massive database of English words, acronyms and common proper nouns to check and correct the user’s input in real time, while also providing them with the ability to ignore suggestions, make corrections one by one, or in large blocks.

The algorithm that suggests corrections Is versatile and accurate, making use of Levenshtein distances to provide variable-length arrays of suggestions as output, all sorted by compatibility and length. Java’s default ugly JButtons are replaced by ones I individually designed, giving the program a modern feel and implementation. You can see the entirety of the code included in this project at my Github page.
